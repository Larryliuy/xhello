# live 直播网站PC端
## 软件架构说明
    描述：总体来说算是一个网页版的YY吧
    软件架构：p2p架构
    服务端：php+nginx
    前端：react技术栈+webRTC+webAudio+webSocket
    网址：https://a701.xtell.cn/softwares/xtell_projects_dev/24_YUN_VIDEO/src/web/index.php
